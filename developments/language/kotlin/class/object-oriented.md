# 객체 지향

## 객체지향의 대두

* 소프트웨어의 복잡성은 시간이 지날수록 증가해옴.
* 객체 및 엔티티와 현실 사이에 유사점을 통해 객체 관점에서 모델링 함.
* 각 객체는 조그마한 컴퓨터로 볼수 있으며  각 객체는 상태와 동작을 가진다.
